#### $How\ to\ download\ LLM\ models\ locally\ for\ ex:\ (deepseek-r1:1.5b)$
```
ollama run deepseek-r1:1.5b
```
#### $List\ the\ avaliable\ LLM\ model\ in\ your\ local\ PC$
```
ollama list
```
